# github-final-project
This repository contains a simple Python script for calculating simple interest. It is part of a micro-finance initiative switch from SVN to Git aimed at empowering low-income individuals by promoting accessible financial tools and transparency. It has an Apache 2.0 license, a code of conduct, donation rules, and best practices for version control.

## Project Purpose
This project serves as:
- A demonstration of basic financial computation (Simple Interest)
- A beginner-friendly open-source project
- A practical example of transitioning from SVN to Git
- A foundation for collaborative development

## Formula Used
Simple Interest is calculated using:

SI = (P × R × T) / 100

Where:
- P = Principal amount
- R = Rate of interest
- T = Time (in years)

## Getting Started

### Prerequisites
- Python 3.14.2 installed

### Run the Script
python simple_interest.py
